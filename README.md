# homebrew-localwispr

Homebrew tap for [LocalWispr](https://github.com/srikruthreddy7/LocalWispr) — local voice dictation for macOS.

## Install

```bash
brew tap srikruthreddy7/localwispr
brew install --cask localwispr
```

## Update

```bash
brew upgrade --cask localwispr
```

## Uninstall

```bash
brew uninstall --cask localwispr
```
